OPEN PORT SCANNER on Golang

Instruction how to install on linux/termux:

pkg install golang

git clone https://github.com/danyprotop/GOFPS-main/

cd GOFPS-main

go mod init gofps

go get github.com/fatih/color

touch ranges.txt

touch found_ips.txt

put the ip ranges in ranges.txt

go run app.go



Instruction how to install on Windows 10/11:

Download golang from https://go.dev/doc/install !!!IT MUST BE IN PATH!!!

open cmd    and I advice you to install gofps in downloads or desktop

git clone https://github.com/danyprotop/GOFPS-main/

cd and way to your GOFPS-main

go mod init gofps

go get github.com/fatih/color

and make ranges.txt and found_ips.txt in GOFPS-main directory

put the ip ranges in ranges.txt

go run app.go
